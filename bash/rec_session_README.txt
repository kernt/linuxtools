How do I use this script?

    Open a terminal.
    Open Tab.
    Run script as:

    ./script.sh foo.ogv
    Switch to tab # 1 and run your command.
    When done switch back to tab #2 and press CTRL + C to save recording to foo.ogv.
    To run foo.ogv type:

    gnome-open foo.ogv
    OR
    mplayer foo.ogv
